# TextBingoGame
- 텍스트로하는 AI와 대결하는 빙고게임 입니다.
1. AI 모드 선택 (Easy 모드, Hard 모드)
2. Easy 모드일 경우 선택 안된 숫자 목록을 만들어서 그 목록 중 하나를 선택하도록 설정
3. Hard 모드일 경우 현재 숫자 중 빙고 줄의 완성 가능성이 높은 줄을 찾아서 그 줄에 있는 숫자 중 하나를 *로 만들어주도록 설정
4. 빙도 줄이 5개가 먼저 되면 승리
